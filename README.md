# xaru
xaru는 marumaru.in의 만화를 일괄로 다운받아주는 커맨드 프로그램입니다.

## Using
[도움말](https://xaru.ga)

## Build
 - Need: DMD 2.6.x~, DUB 1.x~
 
```
git clone https://github.com/zhanitest/marumaru.d.git
cd marumaru.d
dub add-local ./
cd ..

git clone https://github.com/zhanitest/xaru.git
cd xaru
dub run
```
